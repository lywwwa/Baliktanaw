The asset contains 2 versions:


- a classic version (frozen/non-animated foliages)

- an AFS (Advanced Foliage Shader v.4) version (moving/animated foliage)



To be able to use the AFS prefabs,

the "Advanced Foliage Shader V.4" (by Forst) asset is REQUIRED

and imported in your project before this asset.



You can get the AFS v.4 asset there:

https://www.assetstore.unity3d.com/en/#!/content/3253
